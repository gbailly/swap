Welcome to SW@P


SW@P is a javascript library aiming at reinforcing the privacy during transactions made on the internet. It provides a way for its users to interact with organizations in an efficient and anonymous manner. The system you're now interacting with has been built on top of the library to demonstrate its functionalities. This system operates in three steps described below.


Learn more »

1. Setting up the environment

This first step will let you generate a new secret key if you don't already have one. This key will be used in the following steps to ensure the privacy of your communication with the server.

Learn more »

2. Issuing a new credential

In this second step, you will interact anonymously with our server to receive a credential on a set of attributes you provide.


Learn more »

3. Proving possession

This last step consists of proving you indeed possess a credential issued by our server. The authenticity of your credential will be checked to ensure it has not been tampered with.
